# BEX SCM 대시보드 통합 구축 지시서 (Phase 2)

> **작업 의뢰자**: 신용수 (마케팅팀장)
> **작업 도구**: Claude Code
> **대상 프로젝트**: bex-scm (BEX SCM BI v3.1)
> **작업일**: 2026-05-13 (갱신: 2026-05-15)
> **문서 버전**: v2.0 (기존 대시보드 + SCM 화면가이드 통합)
> **목적**: SCM 5개 화면을 BEX 대시보드에 통합하여 단일 운영 시스템 구축

---

## 1. 프로젝트 목표

기존 BEX 대시보드는 매출/이익/미수 시각화에 특화되어 있고, 운영 데이터(상품·거래처·BOM·발주)는 별도 관리되고 있다. 이를 BEX 대시보드 안으로 통합하여 **단일 시스템에서 모든 운영 데이터를 관리**한다.

SCM 5개 화면 설계를 BEX 대시보드 안에 SCM 섹션으로 신규 구축한다.

## 5. SCM 화면 1 — 상품마스터

**경로**: `/scm/products`
**용도**: 완제품 마스터 — 호수/색상별 단위로 관리

### 5-1. 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│ 🏷️ 상품마스터 [검색바] [카테고리▼] [상태▼] [+ 추가] [엑셀↓] │
├─────────────────────────────────────────────────────────┤
│ ┌─ 좌측 테이블 (50%) ──────┬─ 우측 상세 패널 (50%) ────┐ │
│ │ 통합품번 │ 상품명 │ 바코드 │ [기본정보][사양][문서][원가] │ │
│ │  ...     │   ...  │   ...  │                            │ │
│ │  ⏵ 행 클릭 시 우측 표시  │ ← 클릭한 상품 상세 표시      │ │
│ └──────────────────────────┴──────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

### 5-2. 좌측 테이블 컬럼

- 통합품번 (UID)
- 상품명 (한국어)
- 바코드 (88…)
- 카테고리, 시리즈, 제조사
- 호수/색상 (개별 단위)
- **상태**: 진행 / 단종예정 / 런칭예정 / 단종 (한국어 칩)
- 원가
- 활성/비활성

### 5-3. 우측 패널 탭

| 탭 | 내용 |
|---|---|
| 기본정보 | 이름, 바코드, 카테고리, 시리즈, 제조사, 호수, 색상, 상태, 원가, 메모 |
| 사양 | 박스/인박스/아웃박스 치수·무게, 용적, 출시일 등 |
| 문서 | 제품 관련 첨부 (디자인 가이드, 인증서 등) |
| 원가이력 | 시점별 원가 변경 이력 (BOM 활성 전환과 연동) |

### 5-4. 핵심 기능

- 검색 (이름/바코드)
- 카테고리·상태 필터
- 엑셀 업/다운 (한국어·영문 상태 양쪽 호환)
- 우측 패널 인라인 편집

## 6. SCM 화면 2 — 거래처

**경로**: `/scm/vendors`
**용도**: 제조사·부자재·소모품 업체 통합 관리 + 다중 담당자

### 6-1. 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│ 🏢 거래처 [검색][역할▼][비활성포함☐] [+ 추가] [엑셀↓]    │
├─────────────────────────────────────────────────────────┤
│ ▶/▼ 거래처명 │ 역할      │ 사업자번호 │ 담당자 │ ... 액션 │
│ ▶  (주)코스맥스│🏭제조 📦부자재│ 123-..   │ 홍길동+1 │ 수정 │
│ ▼  (주)비엘비  │🏭제조       │ 456-..   │ 김영업  │       │
│      └ 박마감 (영업 담당) email phone                     │
│      └ 이대표 (비활성)                                     │
└─────────────────────────────────────────────────────────┘
```

### 6-2. 컬럼

- 거래처명 (다중 담당자 시 "담당 N" 배지)
- **역할 태그** (다중 선택 가능): 🏭 제조 / 📦 부자재 / 📮 소모품
- 사업자등록번호
- **첫 담당자** 이름 (펼치면 나머지 보임)
- 이메일, 전화, 메모, 활성 상태

### 6-3. 다중 담당자 모델

- 거래처 1곳 → 담당자 N명 (영업/마감/대표메일 등)
- 펼치기 토글로 추가 담당자 확인
- 거래처 자체 활성 + 담당자별 활성 분리

### 6-4. 거래처 추가/수정 폼

- 거래처명, 사업자번호
- 역할 (다중 체크): 제조사 / 부자재 업체 / 소모품 업체
- **담당자 인라인 테이블** (행 단위 추가/삭제):
  - 이름, 이메일, 전화, 메모/역할(영업·마감 등), 활성 체크
- 거래처 메모
- 비활성화 토글

### 6-5. 엑셀 다운

- 거래처별 담당자 행 펼침 (담당자 N명 → N행, 각 행에 거래처 정보 반복)
- 검색·정렬·복붙 편의

## 7. SCM 화면 3 — BOM 관리

**경로**: `/scm/boms`
**용도**: 제품 × 부자재 구성 관리 + MOQ별 단가 + 활성 BOM 자동 원가 갱신

### 7-1. 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│ 📋 BOM 관리 [상품 검색] [+ 신규 BOM]      [엑셀↓]        │
├─────────────────────────────────────────────────────────┤
│ ┌─ 제품 카드 1 ────────────────────────────────────────┐│
│ │ S-EGF 리프팅 부스트샷 (8809603934628)                ││
│ │   [BOM ver1 ★활성] 5,000 / 10,000 / 20,000 (MOQ)     ││
│ │     - 단상자  (코스맥스) 5g     ₩2,921 ₩2,500 ₩2,000  ││
│ │     - 봉합라벨 (영오) 2개      ₩50    ₩45    ₩40    ││
│ │     - 인박스   (다우)          ₩100   ₩90    ₩80    ││
│ │     ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ ││
│ │     합계               ₩3,071  ₩2,635 ₩2,120          ││
│ │   [BOM ver2 (비활성)]   …                              ││
│ └────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────┘
```

### 7-2. 구성 요소

- **제품 카드**: 한 제품의 모든 BOM 버전을 표시
- **BOM 카드** (제품당 N개 가능):
  - 버전명 (예: v1, v2, "2024 패키지")
  - 활성 마크 ★ (제품당 1개만 활성, ProductMaster.원가에 자동 반영)
  - **MOQ별 단가 컬럼** (5천/1만/2만 등 단계별 단가)
  - 부자재 행들
- **부자재 행**:
  - 부자재명, 업체, 용량
  - 소요량 (제품 1개당)
  - MOQ별 단가 (BOM 전용 단가 우선, 없으면 부자재 마스터 단가 폴백)
- **합계 행**: MOQ별 BOM 총 원가

### 7-3. 핵심 기능

- BOM 신규 생성 (제품 선택 → 부자재 인라인 추가 → MOQ 단가 입력)
- BOM 복사 (이전 버전 그대로 가져와 단가만 수정)
- 활성 BOM 전환 → ProductMaster.원가 자동 갱신
- **MOQ별 단가**: 같은 부자재가 다른 BOM에서 다른 단가일 수 있음 (별도 단가 컬럼 5개)
- 부자재 인라인 등록 (새 부자재 + BOM 추가를 한 폼에서)
- 발주에 사용된 BOM 삭제 차단 (비활성화 권유)
- 엑셀 다운: 임포트 양식과 동일 (재임포트 가능)

### 7-4. 더미 데이터

- WD-40 100ml BOM v1 (캔·노즐·라벨)
- 케이블타이 200mm BOM v1
- 방진복 XL BOM v1

## 9. SCM 화면 5 — 발주 관리 (가장 중요)

**경로**: `/scm/purchase-orders`
**용도**: 발주서 작성·이력 관리·입고 기록·PDF/엑셀 출력

### 9-1. 발주 이력 (목록 페이지)

**경로**: `/scm/purchase-orders`

```
┌─────────────────────────────────────────────────────────┐
│ 🛒 발주 관리 [검색][상태▼][거래처▼][기간] [+신규] [엑셀↓] │
├─────────────────────────────────────────────────────────┤
│ PO번호 │ 발주일 │ 거래처 │ 품목명 │ 수량 │ 단가 │ 납기 │ │
│ PO2605000001│5/12│코스맥스│S-EGF…│5,000│₩2,921│5/30  │ │
│ PO2605000001│5/12│코스맥스│봉합라벨│5,000│₩50  │5/30  │ │
│   ⏵ PO번호 클릭 → 상세 페이지로 이동                      │
└─────────────────────────────────────────────────────────┘
```

**컬럼**: PO번호 (자동: PO + YYMM + 6자리) / 발주일 / 거래처 / 품목명 (상품/소모품 모두 한 줄) / 수량 / 단가 / 합계 / 납기일 / 입고예정일 / 입고일 / 입고수량 / **상태** (진행·종결·취소·**잔량**: 수량 - 입고)

**필터**: 상태 필터 (진행/종결/취소) / 거래처 필터 / 기간 검색 (발주일·납기일·입고일 3모드) / 통합 검색 (품목명/바코드/PO번호/거래처)

입고 기록 있으면 행 펼침 (입고 N건 = N행)

## 10. 공통 화면 패턴

5개 화면 모두 적용:

### 10-1. 인증·권한

- 모든 SCM 페이지: 기존 BEX 대시보드의 `AuthGate` 컴포넌트 사용 (별도 인증 추가 없음)
- 미인증 → `/login` 리다이렉트

### 10-2. 액션

- **추가/수정**: 폼 팝업
- **삭제**: 확인 다이얼로그 + 참조된 데이터 있으면 비활성화 권유
- **활성/비활성**: 토글 + 비활성 항목은 흐리게 표시
- **엑셀 다운**: 모든 화면에 다운 버튼 (필터 조건 반영)

### 10-3. 검색·필터

- 통합 검색바 (이름/주요 컬럼 합쳐서)
- 카테고리·역할·상태 등 드롭다운 필터
- 비활성 포함 토글
- 결과 카운트 표시 (예: "총 38곳")

### 10-4. 모바일/접근성

- 반응형 레이아웃 (테이블 가로 스크롤)
- 한국어 라벨 (예: 상태값 "진행", "단종예정" 등)

### 10-5. 디자인

- **현재 적용된 디자인 시스템 그대로** 유지
- 기존 컴포넌트 `PageHeader`, `KpiCard`, `ChartCard`, `LoadingState`, `ErrorState` 재사용
- 카드/테이블/버튼 스타일은 기존 페이지(`/profit`, `/analysis/category` 등) 참고
- 폰트/색상/간격 일관성 유지
